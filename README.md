# tfl_app
TfL Fare Analyser is a simple analytics tool that helps London commuters
understand whether their TfL travel choices are cost-effective.

Users can upload their TfL journey history and see:
- total spend over a selected period
- an estimated comparison between PAYG fares and travelcards
- a clear recommendation on which option would have been cheaper

This project focuses on data analysis, cost optimisation logic, and
turning raw journey data into actionable insights.
